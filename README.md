# Go_to_space-THREEJS

## Les contrôles

- **Avancer :** Z
- **Reculer :** S
- **Gauche :** Q
- **Droite :** D
- **Caméra tourne haut :** Flèche haut ↑
- **Caméra tourne bas :** Flèche bas ←
- **Caméra tourne gauche :** Flèche gauche ↑
- **Caméra tourne droite :** Flèche droite →
- **Monter :** Barre espace
- **Descendre :** Ctrl gauche
- **Boost vitesse :** Majuscule gauche
